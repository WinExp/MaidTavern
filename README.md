# Maid Harvest Grape
This is a Touhou little maid addon which adds a maid task makes maids can harvest grapes in Kaleidoscope: Tavern.

Maids will only harvest grapes that are 3 blocks above the ground, so you should place the trellis 4 blocks above the ground.
